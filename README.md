# 🌱 Projeto Estufa Do Joãozinho - Formação Agrinho 2026

Site interativo desenvolvido para o Concurso Agrinho 2026 com o tema: **"Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente"**

## 📋 Funcionalidades do Site

### 1. **Simulador de Estufa de Morango** 🎛️
   - Controle interativo de três variáveis ambientais:
     - **Temperatura**: ajuste de 0°C a 40°C
     - **Umidade do Ar**: controle de 0% a 100%
     - **Umidade do Solo**: ajuste de 0% a 100%
   - **Validação automática** das condições ideais para morango:
     - Temperatura ideal: 15°C a 25°C
     - Umidade do ar ideal: 60% a 80%
     - Umidade do solo ideal: 60% a 75%
   - **Feedback visual** em tempo real:
     - ✅ **Status Bom** (verde): Condições sustentáveis e eficientes
     - ⚠️ **Status Alerta** (amarelo): Um ajuste necessário
     - ❌ **Status Perigo** (vermelho): Múltiplos problemas
   - Mensagens educativas que orientam o usuário sobre os problemas e soluções

### 2. **Seção de Notícias** 📰
   - **4 artigos informativos** sobre:
     - Estufas inteligentes e tecnologia IoT na agricultura
     - Técnicas de economia de água (irrigação por gotejamento)
     - Inovações do Concurso Agrinho 2026
     - Certificação de produção sustentável
   - Cards apresentados com data de publicação
   - Conteúdo focado em tecnologia, sustentabilidade e inovação agrícola

### 3. **Informações sobre o Agrinho** 🏆
   - Descrição do Programa Agrinho
   - Tema e objetivos do concurso 2026
   - Explicação sobre saúde, segurança pessoal e ambiental
   - Destaque para o uso de tecnologia na otimização de recursos naturais
   - Espaço para imagem do concurso

### 4. **Sobre Mim** 👤
   - Informações pessoais do desenvolvedor
   - Motivação e visão sobre agricultura sustentável
   - Contexto do projeto e seus objetivos educacionais

### 5. **Navegação Dinâmica** 🗂️
   - Menu com 4 abas principais
   - Navegação via botões interativos
   - Alternância visual entre seções sem recarregar a página
   - Design responsivo com destaque visual da página ativa

## 🎯 Objetivos do Projeto
- Demonstrar como a tecnologia pode otimizar o uso de recursos naturais na agricultura
- Educar sobre as condições ideais para cultivo de morangos em estufa
- Promover práticas agrícolas sustentáveis e eficientes
- Fomentar a inovação tecnológica voltada para o meio ambiente

## 💻 Tecnologias Utilizadas
- **HTML5**: Estrutura semântica
- **CSS3**: Estilização responsiva
- **JavaScript**: Lógica interativa do simulador e navegação

## 🚀 Como Usar
1. Abra o arquivo `index.html` no navegador
2. Navegue entre as seções usando os botões do menu
3. Use os controles deslizantes no simulador para ajustar as variáveis
4. Observe o feedback visual e as mensagens de status
